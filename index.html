Maklumat Program

Tentang Program

Program ini bertujuan untuk menyediakan jamuan iftar, sahur dan moreh untuk 25 orang pelajar dan warga Tahfiz di Maahad Tahfiz Al-Quran Lil Muttaqin sepanjang bulan Ramadan.

Setiap tempahan anda akan membantu meringankan beban mereka dan membolehkan mereka memberi tumpuan sepenuhnya kepada ibadah dan pembelajaran Al-Quran.

Konsep Slot Tempahan

1 slot = 1 set berbuka + 1 set sahur + 1 set moreh untuk 1 orang

Nilai: RM50 / seorang / sehari

1 slot (1 orang/hari) RM50
25 slot (25 orang/hari) RM1,250
30 hari Ramadan RM37,500

Siapa Penerima Manfaat?

25 orang pelajar dan warga Tahfiz di Maahad Tahfiz Al-Quran Lil Muttaqin yang akan menerima manfaat langsung dari program ini.

Mereka adalah golongan yang berdedikasi untuk menghafaz dan mempelajari Al-Quran, dan tempahan anda akan membantu mereka dalam perjalanan ini.
