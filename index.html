Panduan Setup Google Sheets

Langkah 1: Buat Google Sheets Baru

1. Pergi ke Google Sheets

2. Klik "Blank" untuk membuat spreadsheet baru

3. Namakan spreadsheet sebagai "Sumbangan Ramadan 2026"

4. Salin ID spreadsheet dari URL (contoh: https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit)

Langkah 2: Publish Spreadsheet

1. Klik "File" > "Share" > "Publish to web"

2. Dalam tab "Link", pilih "Entire Document"

3. Klik "Publish" dan kemudian "OK"

4. Salin ID spreadsheet dari URL

Langkah 3: Dapatkan API Key

1. Pergi ke Google Cloud Console

2. Buat projek baru atau pilih projek sedia ada

3. Enable Google Sheets API

4. Buat API Key dengan kebenaran yang sesuai

5. Pastikan tiada sekatan pada API Key

6. Salin API Key

Langkah 4: Setup Halaman Web

1. Masukkan ID spreadsheet dan API key di ruangan yang disediakan

2. Pilih mod "Baca Sahaja" jika API Key anda mempunyai sekatan

3. Klik "Simpan & Sambung"

4. Tunggu beberapa saat untuk sambungan dibuat

5. Halaman web akan dimuatkan dengan data dari Google Sheets

Mod baca sahaja dipilih. Data tidak akan disimpan ke Google Sheets.
